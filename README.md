# PHP掲示板

簡単な PHP 掲示板のサンプルプロジェクトです。

## 機能

- 会員登録
- ログイン / ログアウト
- 投稿の削除
- 投稿一覧の表示
- ログインから一覧ページまでのローディング画面

## 特徴

- 参考資料をもとに作成しています
- デザインはオリジナルに変更しています
- 今後とも、更新していきます。

## 使用方法

1. PHP 環境（XAMPP や MAMP など）にプロジェクトを配置
2. データベースを作成し、`config.php` で接続情報を設定
3. ブラウザでアクセスして利用

## 注意事項

- データベース接続情報はセキュリティに注意してください
- サンプルプロジェクトとして作成しているため、本番環境での使用は推奨しません
