# PHP掲示板

簡易的な 会員制PHP掲示板アプリ のサンプルプロジェクトです。
ユーザー登録・ログイン機能を備え、ログインユーザーのみが投稿・削除を行える構成になっています。

## 機能

- 会員登録
- ログイン / ログアウト
- 投稿の削除
- 投稿一覧の表示
- ログインから一覧ページまでのローディング画面

## 使用技術
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- MAMP

## 特徴

- 参考資料をもとに作成しています
- デザインはオリジナルに変更しています
- ログインユーザーのみ投稿・削除が可能な設計
- 投稿削除は 投稿者本人のみ可能 に制御
- 今後とも、更新していきます。

## 使用方法

1. PHP 環境（XAMPP や MAMP など）にプロジェクトを配置
2. データベースを作成し、`config.php` で接続情報を設定
3. ブラウザでアクセスして利用

## 注意事項

- データベース接続情報はセキュリティに注意してください
- サンプルプロジェクトとして作成しているため、本番環境での使用は推奨しません
